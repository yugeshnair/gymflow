# GymFlow

A multi-user gym workout tracker built to log workouts, track weekly training goals, and suggest what to train next based on your recent activity.

## Features (in progress)
- Google Sign-In authentication
- Log workouts (date, muscle group, duration, notes)
- Weekly progress tracking
- Smart suggestion engine (rule-based)

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: SQLite (moving to PostgreSQL)
- Deployment: AWS
